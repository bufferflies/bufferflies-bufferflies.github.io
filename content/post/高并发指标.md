# 高并发指标
- RT 响应时间 
- throughput： 吞吐量
- QPS：question per second 每秒的请求数量
- 并发用户数
- PV：page view 页面被浏览的次数(点击量)
- TPS：Transactions Per Second 每秒的事务数
- RPS：Responses Per Second 每秒处理响应数 并发数/平均响应时间
- QPS: Query Per Second 每秒可以处理的请求数
- TP90: 超过90%的请求处理时间
